# Gizmapps MOOC blog
## My digital Education Story

Gizmapps MOOC blog is a responsive personal blog, which is made as a portfolio 
project for the codecademy full stack engineer path. The main target of this bloody thing is
to show to potential employers and to myself some knowledge about web technologies (HTML, JS, CSS).


## Sections

#### The project has an index.html page with six divs and a contact.html page with two divs.

the index.html : 
- The first div demonstrates the use of a navigation bar and the use of specific fonts
- The second div demonstrates the use of inline text and images
- The third div demonstrates the use of event handlers with toggle - read more - read less
- The fourth div demonstrates the use of event handlers. the use of accessibility ARIA roles and a css blinking effect
- The fifth div demonstrates the use of mouse events (mousedown, mouseup) on an unordered list and event handlers
- The sixth div demonstrates the use of event handlers and how to add-remove them , some java script logic, css hover effect

the contact.html : 
- The first div demonstrates the use of a navigation bar.
- The second div demonstrates the use of inline icons for further usage such as links
